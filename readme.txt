Create pages in WordPress with ease using drag drop interface.

== Description ==

You can use this plugin to create from a simple to advanced pages in WordPress with ease. 

All pages created are responsive and mobile friendly.

There are so many flexibilities in the plugin. You can re-order the layout of rows, columns in any way you want.

The plugin supports many Google fonts natively. You will never run out of good fonts to use on your pages.


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= How do I create a page? =

Go to Visual Editor->Create page to start creating a page.

= How do I edit a page? =
Go to Visual Editor->Create page. Look to top right, click on Edit->Pages then select the page you want to edit.

= How do I save a page? =

In Create Page window, click on Save (top right). You have two options: Draft and Publish.

= How do I view a page? =
After creating the page, you click on Edit->View to view your page. Make sure you save the page first.
